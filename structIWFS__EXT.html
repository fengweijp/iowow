iwrc(* IWFS_EXT::add_mmap) (struct IWFS_EXT *f, off_t off, size_t len)

Register an address space specified by off and len as memory mmaped region within this file.

It is not required for this region be physically represented in the file's address space. As soon as this region will be used for reading/writing it will be mmaped and direct mmaped memory access will be used for IO in this area.

For example:

f.add_mmap(&f, 10, 20);
f.read(&f, 5, buf, 10, sp); //read [5-15) bytes
// [5-10) bytes will be read using system `pread`
// [10-15) bytes will be retrieved by direct `memcpy` from mmapped region

Pointer to this region can be retrieved by IWFS_EXT::get_mmap

Parameters
fIWFS_EXT
offOffset of mmaped region
lenLength of mmaped region
Returns
0 on success or error code.
