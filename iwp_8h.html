enum iwp_lockmode

File locking mode.

File locking mode acquired by process opened this file.

Enumerator
IWP_NOLOCK 

No lock on file.

IWP_RLOCK 

Acquire read lock on file.

IWP_WLOCK 

Acquire write lock on file.

IWP_NBLOCK 

Do not block current thread if file have been locked by another process.
