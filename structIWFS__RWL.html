iwrc(* IWFS_RWL::lock) (struct IWFS_RWL *f, off_t off, off_t len, iwrl_lockflags lflags)

Acquire a lock over the address range specified by off and len.

Lock type is specified by iwrl_lockflags flags. If the given address range intersects with some other write locked range owned by another thread the current thread be blocked until that lock is held.

Parameters
fIWFS_RWL
offLocked address space offsets.
lenLength of locked spaces.
lflagsA locking typem either IWRL_READ or IWRL_WRITE
Returns
0 on success or error code

iwrc(* IWFS_RWL::try_lock) (struct IWFS_RWL *f, off_t off, off_t len, iwrl_lockflags lflags)

Try to acquire a lock over the address range specified by off and len.

Lock type is specified by iwrl_lockflags flags. If the given address range intersects with some other write locked range owned by another thread the call will fail with IW_ERROR_FALSE error code.

Parameters
fIWFS_RWL
offLocked address space offsets.
lenLength of locked spaces.
lflagsA locking typem either IWRL_READ or IWRL_WRITE
Returns
0 on success or error code
