Parameters

lenLength in bytes of locked space.

Try to acquire a lock for the address range specified by start and len. If lock cannot be acquired without waiting this function returns IW_ERROR_FALSE error code as response status.

Parameters
lkIWRLOCK pointer.
startOffset of the first byte of locked address space.
lenLength in bytes of locked space
flagsLocking flags. 0 - for read locks otherwise write lock
Returns
0 on success, IW_ERROR_FALSE if lock cannot be acquired without waiting, or error code.
